欢迎使用AI模型调用平台

本平台支持在同一页面内调用DeepSeek、Google Banana和Meshy三种AI模型，您可以配置API地址和密钥，发送请求并查看响应结果。

本页面发起的是真实网络请求（非模拟响应）。请注意：浏览器直连第三方 API 可能受到 CORS 限制，且密钥会暴露在前端环境，生产环境建议通过后端代理转发。

使用帮助

基本操作

  • 在左侧选择要使用的AI模型
  • 在配置区域填写API地址和密钥
  • 在输入区域填写提示词或上传参考图像
  • 点击发送请求按钮调用API
  • 在输出区域查看响应结果

模型说明

DeepSeek

大语言模型，支持文本生成、对话等功能。

Google Banana

图像生成模型，支持文本到图像、图像编辑等功能。

Meshy

3D模型生成模型，支持文本到3D、图像到3D等功能。

API配置

您需要获取各模型的API密钥才能使用本平台：

  • DeepSeek: https://api.deepseek.com
  • Google Banana: https://ai.google.dev/
  • Meshy: https://www.meshy.ai/zh/api
